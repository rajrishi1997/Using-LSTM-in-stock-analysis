## Using-LSTM-in-stock-analysis
Instructor: Prof. Amit Sethi, IIT Bombay  
Extracted the company data from NY finance, preprocessed it & performed data analysis   
Applied Long short-term memory (LSTM) model with 3 added layers and dropout regularization to predict the closing stock-prices and visualized the training and validation loss of the model.  
Achieved an R2-score of 0.94 and MSE of 0.05.
